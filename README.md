# website-uji-level-non-dicoding

A minimalist React project initialized with Vite.

## Tech Stack
- **Framework:** React + Vite
- **Styling:** Tailwind CSS
- **Routing:** React Router DOM

## How to run (For Beginners)

Jika Anda baru pertama kali melakukan clone pada project ini, ikuti langkah-langkah berikut:

1. **Clone repository ini ke komputer Anda:**
   Buka terminal atau command prompt, lalu jalankan perintah berikut (ganti URL_REPOSITORY_ANDA dengan URL repository GitHub yang sebenarnya):
   ```bash
   git clone URL_REPOSITORY_ANDA
   ```

2. **Masuk ke dalam folder project:**
   ```bash
   cd website-uji-level-non-dicoding
   ```

3. **Install semua dependensi (packages) yang dibutuhkan:**
   Pastikan Anda sudah menginstall [Node.js](https://nodejs.org/) di komputer Anda. Lalu jalankan:
   ```bash
   npm install
   ```

4. **Jalankan local development server:**
   ```bash
   npm run dev
   ```

5. **Buka di Browser:**
   Setelah server berjalan, biasanya akan muncul URL seperti `http://localhost:5173`. Klik link tersebut atau copy-paste ke browser Anda untuk melihat hasilnya.

